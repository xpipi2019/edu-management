# 教育管理系统

一个基于 Vue 3 + TypeScript + Element Plus 开发的现代化教育管理系统，提供完整的学校教务管理功能。

## 功能特性

### 🎓 核心模块
- **用户管理**：支持学生、教师、管理员多角色管理
- **课程管理**：课程信息、开课计划、排课管理
- **成绩管理**：教师录入成绩、学生查询成绩
- **学籍管理**：学生状态跟踪、奖惩记录管理
- **教室管理**：教室资源调度和使用管理

### 🛡️ 系统特色
- **权限控制**：基于角色的细粒度权限管理
- **响应式设计**：支持多设备访问
- **现代化UI**：基于 Element Plus 的美观界面
- **Mock数据**：内置完整的模拟数据支持开发调试

### 👥 角色职责

| 用户类型   | 主要职责                   | 核心权限                                                         |
| ---------- | -------------------------- | ---------------------------------------------------------------- |
| 超级管理员 | 系统整体管理与安全维护     | 用户管理、角色权限分配、部门管理、系统配置、日志管理             |
| 教务人员   | 课程排课与学籍管理         | 课程管理、开课管理、排课、教室分配、成绩审核、学籍异动管理       |
| 教师       | 课程教学与学生管理         | 课程信息维护、开课管理、成绩录入与查询、选课审核、排课查询       |
| 学生       | 学习与个人信息管理         | 在线选退课、成绩查询、奖惩查询、学籍状态查询、个人资料维护       |

## 系统架构

### 📊 数据库设计

**数据库基本信息**：
- 数据库名称：`edu_management_database`
- 字符集：`utf8mb4` (支持完整Unicode字符)
- 兼容性：MySQL 5.7+, MariaDB

**核心表结构**：
- **用户权限系统**：user, role, permission, user_role, role_permission
- **组织架构**：department（支持多级部门结构）
- **人员管理**：teacher, student
- **教务管理**：course, course_offering, enrollment, schedule
- **成绩管理**：grade（支持自动计算总评成绩和绩点）
- **学籍管理**：student_status, reward_punishment
- **设施管理**：classroom

**高级功能**：
- 📈 **智能视图**：自动聚合学生信息、课程选课率统计
- ⚡ **触发器**：自动维护数据一致性，计算成绩和绩点
- 📋 **存储过程**：生成学生成绩单、计算GPA等业务逻辑

### 🏗️ 系统模块

| 模块类别   | 功能模块     | 主要功能                                                     |
| ---------- | ------------ | ------------------------------------------------------------ |
| **管理员** | 用户管理     | 用户增删改查、角色分配、权限管理、部门管理                   |
|            | 系统管理     | 参数配置、日志管理、数据备份                                 |
| **教务**   | 课程管理     | 课程信息管理、开课审批、排课管理、教室分配                   |
|            | 学籍管理     | 学生注册、异动管理、奖惩记录、成绩审核                       |
| **教师**   | 教学管理     | 课程信息维护、成绩录入、选课审核                             |
|            | 查询功能     | 排课查询、学生管理                                           |
| **学生**   | 选课管理     | 在线选课、退课、课程表查询                                   |
|            | 信息查询     | 成绩查询、奖惩记录、学籍状态、个人信息维护                   |

## 技术栈

- **前端框架**：Vue 3 + Composition API
- **类型支持**：TypeScript
- **UI 组件库**：Element Plus
- **状态管理**：Pinia
- **路由管理**：Vue Router 4
- **构建工具**：Vite
- **代码规范**：ESLint + Prettier
- **测试框架**：Vitest + Playwright

## 快速开始

### 环境要求
- Node.js 18+ 
- npm 或 yarn 或 pnpm

### 安装依赖
```bash
cd frontend
npm install
```

### 开发环境运行
```bash
npm run dev
```
访问 http://localhost:3000

### 生产环境构建
```bash
npm run build
```

### 其他命令
```bash
# 类型检查
npm run type-check

# 代码检查
npm run lint

# 代码格式化
npm run format

# 单元测试
npm run test:unit

# E2E测试
npm run test:e2e
```
